# -Projeto-Integrador-Pra-a-Conectada-Infraestrutura-de-Redes
Projeto Integrador de Redes “Praça Conectada”: infraestrutura com VLANs (Usuários, DMZ e Servidores), roteamento Inter-VLAN (Router-on-a-Stick), redundância WAN e serviços DHCP, DNS e Web. Implementado em Dual-Stack (IPv4/IPv6), com arquitetura escalável para evolução contínua durante a graduação em redes de computadores.
